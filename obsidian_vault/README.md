# AI株式会社：公的ナレッジ保管庫 (Obsidian Vault)

このディレクトリは、AI（Antigravity）と President の共同活動によって生成された全てのナレッジが蓄積される場所です。

## 📁 ディレクトリ構成
- `archives/`: 月次の教訓ログなどが保存されます。
- `meetings/`: 会議の議事録（将来的にここへ出力するように設定可能）。
- `research/`: サブAIが調査した専門資料。

---
**Presidentへ**:
Obsidianをインストール後、この `obsidian_vault` フォルダを「既存のフォルダを保管庫として開く」で選択してください。
すると、ここにある全ての教訓がリンクされ、グラフとして閲覧できるようになります。
