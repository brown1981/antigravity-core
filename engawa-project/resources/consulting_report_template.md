# 【縁側サイクル】沖縄住宅向け 排熱・床下除湿システム 導入提案書

## 1. はじめに（ビジョン）
- 「捨てていた熱を、資産価値と快適な暮らしに変える。」
- 背景：沖縄特有の高温多湿な環境と、サーバー/ASIC機器から排出されるクリーンな熱の有効活用。

## 2. 現状の課題分析（Research by COO）
- 沖縄の木造・RC住宅における床下湿度のリスク（シロアリ、カビ）。
- 除湿機導入による電気代の負担。

## 3. システム導入のメリット（CMO Focus）
- **コストゼロの熱源**: 24時間稼働するマイニング機器からの排熱（約70度）を利用。
- **恒久的な除湿環境**: 物理的な熱送風による強制乾燥。
- **資産価値の保護**: 住宅寿命の延命とメンテナンスコストの削減。

## 4. 予想ROI（投資対効果）
- 5万円のコンサルティング費用 vs 年間除湿電気代の削減額。
- 住宅の減価償却資産としての価値維持。

## 5. 技術仕様と安全性
- 非燃焼系熱源の安全性。
- ダクト設計と静音化対策。
