# 【縁側サイクル】沖縄住宅向け 排熱・床下除湿システム 導入提案書 (v1)

## 1. ビジョン
**「捨てていた熱を、資産価値と快適な暮らしに変える。」**

沖縄の過酷な湿度は、住宅の寿命を静かに削っています。一方で、24時間稼働し続けるマイニング機材（Antminer Z15 Pro）は、私たちが制御すれば、住宅を守る貴重な「クリーンな熱源」へと変わります。
私たちは、マイニング報酬を最大化するだけでなく、その副産物である「熱」を宜野湾市の居住空間へ最適化して提供します。

## 2. 沖縄特有の湿度の課題
- **平均湿度 75%**: 宜野湾市を含む沖縄県内では、年間を通じて高い湿度によるカビやダニの発生リスクが極めて高い。
- **床下湿度の脅威**: 換気が滞りやすい床下は、シロアリ被害の温床となります。
- **除湿コスト**: 従来の除湿機導入は、月数千円の電気代とメンテナンスを伴います。

## 3. 「縁側サイクル」のソリューション
- **熱源**: Bitmain Antminer Z15 Pro から発生する 72度の乾燥風。
- **効果**: 床下へ直接送風することで、含水量を低下させ、シロアリが生存不可能な乾燥環境を構築。
- **ランニングコスト 0円**: すでにマイニング収益（月約2.4万円〜）でペイしているエネルギーの再利用。

## 4. 投資対効果 (ROI)
- **導入コンサル費用**: 50,000円 (一括)
- **長期メリット**: 住宅メンテナンス周期（防蟻・防カビ）の延長。資産価値の維持。
- **経済的インセンティブ**: 副産物による利益。
