# 🌙 夜間バッチ処理システム（API単価 50% OFF）ガイド

このフォルダは、**「即時性は必要ないが、作業量が多いタスク」**を深夜に一括処理し、AIの利用コストを半分に抑えるためのハブとなります。

## 1. 運用の流れ
1.  **タスク投入 (日中・夜間)**: 
    - 依頼したい内容をMarkdownファイル（例：`new_article_request.md`）として、この `/batch_tasks/` フォルダ内に保存します。
2.  **バッチ実行 (深夜自動)**: 
    - Antigravityが深夜（例：AM 2:00など）にこれらのファイルを読み取り、`Batch API` を利用して処理を予約します。
3.  **結果確認 (翌朝)**: 
    - 翌朝、処理が完了した結果が `/batch_tasks/completed/` に出力されます。

## 2. 推奨されるバッチタスク例
- **CMO業務**: 10記事分のSEOブログ下書き作成、競合サイトの網羅的リサーチ。
- **CFO業務**: 1ヶ月分のマイニング収益CSVの月次レポート生成。
- **COO業務**: 週末の沖縄県内「補助金・助成金」の全件要約。

## 3. 使い方のルール
- 各ファイルには、**「どのエージェント（CMO, CFO等）に任せるか」**を冒頭に記述してください。
- 50%割引を受けるため、完了までに最大24時間の余裕を持たせてください。
